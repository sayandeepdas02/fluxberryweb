# Fluxberry AI

[![Fluxberry AI](https://fluxberry.ai/opengraph-image.png)](https://fluxberry.ai)

Fluxberry AI is an end-to-end AI Hiring OS that helps startups and growing teams source, screen, interview, and onboard talent faster than ever before.

## Features

- **AI Talent Sourcing**: Search, discover, and engage top candidates instantly.
- **Modern Job Board & ATS**: Create branded career pages and manage applicants through structured hiring pipelines.
- **Interview Intelligence**: Standardize interviews, reduce bias, and accelerate decision-making.
- **Offer & Onboarding Automation**: Close the hiring loop with seamless offer generation and document collection.

## Getting started

Install the dependencies. We recommend using `pnpm`.

```bash
pnpm install
```

Then, start the development server:

```bash
pnpm run dev
```

Visit [http://localhost:3000](http://localhost:3000) in your browser to view.
